# Content Adversarial

This repository manages an automated, multi-agent content generation workflow designed to translate peer-reviewed scientific literature into engaging, fact-checked, and brand-aligned short-form video scripts (TikTok, Instagram Reels, YouTube Shorts).

## Core Focus

The workflow focuses on promoting conscious, healthy, and responsible food consumption, acting as an educational counterweight to ultra-processed foods and unethical supply chains.

## Project Structure

*   **Research & Synthesis Phase:** Pulls research references and metadata from academic articles, aligns with target topics, and compiles findings.
*   **Script Writing Phase:** Drafts conversational short-form scripts incorporating specific brand guidelines and hooks.
